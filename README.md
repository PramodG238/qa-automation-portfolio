# QA Automation Portfolio
 
Automated test suite using Robot Framework + GitHub Actions CI/CD.
 
## Tech Stack
- Robot Framework 7.0
- SeleniumLibrary
- GitHub Actions (CI/CD)
- Python 3.11
 
## Tests
- Login validation (valid / invalid / empty fields)
- Auto-runs on every push via GitHub Actions
 
## How to Run Locally
python -m pip install -r requirements.txt
python -m robot tests/
